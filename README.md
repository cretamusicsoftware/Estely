# Real Estate Website

A modern, responsive real estate platform that helps users buy, sell, and rent properties with ease and transparency. The website includes a search system, featured listings, service workflow, contact CTA, and a newsletter subscription module.

## Features


Highlighted headline showcasing the platform's value.

Users can search for properties based on category, price range, and property type.

Tabbed form for Buy, Sell, and Rent actions.


Brief overview of the platform's purpose and mission.

Showcases efficiency, transparency, and user control in real estate transactions.

Includes a promotional banner with a video-play feature.
